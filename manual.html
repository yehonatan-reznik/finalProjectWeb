Operator Manual – SkyShield Control System

Return to Control Panel

Navigate to the login page and authenticate using your assigned credentials. Upon success, you are redirected to the Control Panel. If access fails, contact the Command Unit.

Press “Scan Mode” to initiate automated sweeps. The camera view displays scanlines while scanning. Press again to return to manual control.

Confirm target and safety status. Press “Fire Laser” to discharge. The camera briefly flashes red and the console logs the event. If safety interlock is active, firing is aborted.

Open the Logs page for a timestamped record of operations. Use the search bar to filter by action or result keywords (e.g., FIRE, SCAN, SUCCESS).

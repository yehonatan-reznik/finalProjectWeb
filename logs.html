How To Read It
  • Target rows Focus on confidence, `x/y`, and the camera move hint. Those are the highest-signal lines during tracking.
  • Detection rows Backend load, profile switches, and clear-state returns show whether the model is behaving normally.
  • Controller rows Use them to match operator input against target movement during bench tuning.
Important Fields
When target exists

Look for `Target detected`, then read `x`, `y`, and `move=` in the same line before deciding on manual camera input.

When target drops

Check whether the page logged `clear`, a controller movement, or a stream issue first. That tells you if the miss was visual, motion-related, or transport-related.

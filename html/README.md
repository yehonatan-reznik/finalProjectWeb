# HTML Pages

This folder contains the browser-facing pages for the SkyShield demo.

- `index.html` is the login page.
- `control.html` is the main operator dashboard.
- `logs.html` shows event records and review tools.
- `manual.html` explains the workflow and safety steps.
- `form.html` is the project request / submission page.

All pages load shared styles from `../css/` and shared logic from `../js/`.
